概要
看護・介護の交代勤務（日勤・夜勤・公休）のシフトを自動で組むためのツールです。スタッフの希望、施設ごとの必要人数、公休数、夜勤回数のバランスといった条件を踏まえてシフト案を生成し、ルール違反はその場で警告します。手作業で何時間もかかっていたシフト作成の負担を、大きく減らすことを狙いました。
制作背景
看護師として働く中で、毎月のシフト作成が大きな負担であることを現場で痛感していました。
希望を汲みつつ、必要人数と公休を満たし、夜勤の偏りもなくす——この"パズル"を人力で解く大変さを知っているからこそ、「条件を入れれば下案が出る」道具を自分で作りました。現場の事情を分かっているからこそ設計できたツールです。
主な機能
  • スタッフごとの希望シフト入力（日付をクリックして選択）
  • チーム設定・スタッフ管理
  • 施設ルールの設定（時間帯ごとの必要人数・公休数）
  • 入浴介助などの係業務の割り当て
  • 条件を踏まえたシフトの自動生成
  • ルール違反のバリデーション（不足・偏りをその場で警告）
  • 日勤人数・夜勤回数の集計でバランスを可視化
  • データはブラウザ内に保存（オフラインでも動作）
使用技術
HTML / CSS / JavaScript（Vanilla JS）
シフト生成・制約チェックのロジックを自前で実装 / データ保存はブラウザのlocalStorage / レスポンシブ対応
