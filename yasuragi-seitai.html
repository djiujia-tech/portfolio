やすらぎ整体院

URL
https://yasuragi-seitai.vercel.app/
制作背景
腰痛専門の整体院を想定した自主制作サイトです。
「マッサージでは戻ってしまう」腰痛に悩む地域の方に、"原因から改善する院"として見つけてもらい、24時間のWEB予約への導線までを一気につなげることを目的に設計しました。予約部分は、自社開発のLINE予約システム「CUE」との連携を想定した導線設計です（本デモでは連携は未実装）。
ターゲット
慢性腰痛・ぎっくり腰・坐骨神経痛・産後の骨盤の悩みを持つ地域住民。整体が初めての方や、ご年配の方も想定しています。
想定する課題
・「揉んでも戻る」腰痛への不安が、来院前に解消されない
・整体＝痛い/怖いのイメージで一歩を踏み出せない
・電話予約のみだと、営業時間外の「予約したい気持ち」を取りこぼす
目的
  • 「腰痛専門」がひと目で伝わるファーストビュー
  • 症状別（慢性腰痛・ぎっくり腰・坐骨神経痛ほか）の入口設計
  • 検査・ソフト施術・セルフケア指導・予約優先の4つの特徴で信頼を伝える
  • 患者さんの声で「初めてでも安心」を補強
  • 24時間WEB予約を想定した予約導線の設計（LINE予約システム「CUE」との連携を想定）
デザイン
清潔感と信頼感を重視した落ち着いた配色。痛みの"しくみ"を説明する構成で、「納得して来院する」流れを意識しました。強い刺激が苦手な方にも安心感が伝わるトーンでまとめています。
使用技術
HTML / CSS / JavaScript
レスポンシブ対応（SP・PC）/ Vercelデプロイ / 24時間WEB予約の導線設計（CUE連携を想定）
