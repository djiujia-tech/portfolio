Nail Atelier RÊVE

URL
https://nail-atelier-reve.vercel.app
制作背景
ネイルサロン業界向けに、サロンオーナー様への提案を想定して自主制作したデモサイトです。
特定の店舗の実在サイトではなく、業界の課題を想定して企画・デザイン・実装まで一人で行いました。
ターゲット
ネイルサロンを利用する20代〜40代女性（自分へのご褒美・特別な体験を求める層）
想定する課題
・予約や問い合わせがInstagramのDMや電話中心で、対応の手間が大きい
・既存の予約掲載サービスは掲載料・手数料が高く、サロン独自の世界観を表現しにくい
目的
  • サロンのブランドイメージを伝えるデザイン
  • MENU・GALLERY・SHOPを通じて来店前の期待値を高める導線
  • 予約への動線をシンプルに設計し、離脱を減らす
デザイン
ゴールド×アイボリーを基調に、上質感・特別感を演出。
ネイルアートのクローズアップ写真を主役にし、繊細なセリフ体のロゴ・タイトルで世界観を統一した。
制作期間
  • 企画・情報設計 1週間
  • デザイン 2週間
  • コーディング 2週間
使用技術
Figma / Vite / HTML / SCSS / JavaScript（Vanilla JS）
レスポンシブ対応（SP・PC）/ カート機能実装 / スクロールアニメーション / Vercelデプロイ
