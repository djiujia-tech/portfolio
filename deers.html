OITA DEERS U15

URL
https://deers-demo.vercel.app
制作背景
大分市を拠点に活動する中学生世代（U15）のバスケットボールチームを想定した自主制作サイトです。
部員獲得・スポンサー募集・試合結果の発信など、実際のスポーツチームが抱える情報発信の課題に応える構成を意識しました。
ターゲット
入部を検討する中学生・保護者、スポンサー候補となる地元企業
想定する課題
・SNSのみでの情報発信では試合結果・練習日程が埋もれてしまう
・体験参加の申し込み窓口が整っておらず、新規部員の獲得につながりにくい
・スポンサー向けの説明資料が不足し、協賛依頼のハードルが高い
目的
  • チームの活動・雰囲気を伝えるブランディングサイトとしての役割
  • 体験申込フォームによる新規部員獲得の導線設計
  • 試合結果・選手紹介ページで保護者・ファンへの情報発信
  • スポンサーページで協賛依頼をスムーズに
デザイン
チームカラーのダークグリーン×イエローゴールドを基調に、スコアボード風のナビゲーションで「スポーツ感」を演出。
エネルギッシュで力強いビジュアルと、情報の見やすさを両立することを意識した。
制作期間
  • 企画・情報設計 1週間
  • デザイン 1週間
  • コーディング 2週間
使用技術
HTML / CSS / JavaScript（Vanilla JS）
レスポンシブ対応（SP・PC）/ スクロールアニメーション / Vercelデプロイ
